# This app is created with the help of:
- Vite for React
- Firebase
- Axios
- Edamam API

To run this app in your environment, follow the instructions below:

## Prerequisites
- Node.js version 14 or higher
- Firebase account with Firestore and Authentication enabled
- Edamam Nutritional Analysis API

## Set up the environment
1. Clone the repository to your local machine or download it as a zip file and unzip it.
2. Navigate to the project directory:
   ```bash
   cd nutrition-app
3. npm install
4. create .env file
## for .env 
- VITE_FIREBASE_KEY=firebase_key
- VITE_APP_ID=EdamamID
- VITE_APP_KEY=EDAMAM_KEY
 ## run the app
- npm run dev
- visit the localhost given
