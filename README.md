this app is created with the help of
vite for react
firebase
axios
edamam api
to run this app in your environment, 
**Prerequisites
node.js version 14 or hgiher
firebase account with firestore and authentication
Edamam nutritional analysis api
**Set up the environmen
clone the repository to your local machine or download as the zip file and unzip it
cd to nutrition-app
npm install 
create .env with the keys
**run environment 
npm run dev
